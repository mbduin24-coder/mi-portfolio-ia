# mi-portfolio-ia

AI tools portfolio: installation and use of Cursor IDE, Claude Code and Codex as part of a selection process

## Tools installed
- **Cursor IDE**: AI-powered code editor based on VS Code, downloaded from cursor.com
- **Claude Code for VS Code**: Extension by Anthropic (11M downloads), installed from Cursor's marketplace
- **Codex – OpenAI's coding agent**: Extension by OpenAI (3.6M downloads), installed from Cursor's marketplace
- **Git**: Version control system, downloaded from git-scm.com

## Steps completed
1. Created a GitHub account and a public repository called mi-portfolio-ia
2. Downloaded and installed Cursor IDE from cursor.com
3. Connected Cursor to GitHub account
4. Installed Claude Code extension by Anthropic
5. Installed Codex extension by OpenAI
6. Installed Git for Windows
7. Cloned the repository into Cursor using the Clone repo feature
8. Opened and edited the README.md file

## Issues encountered and how I solved them
- **Git not found error**: When trying to clone the repository, Cursor showed "command 'git.clone' not found". I solved this by downloading and installing Git from git-scm.com and restarting Cursor.
- **Broken desktop shortcut**: After reinstalling Cursor, the desktop shortcut stopped working. I deleted it and launched Cursor from the Start menu instead.
- **GitHub authentication**: Cursor required a one-time code to connect to GitHub. I copied the code, pasted it on GitHub's authorization page, and successfully linked both tools.
